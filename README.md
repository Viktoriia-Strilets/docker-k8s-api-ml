# docker-k8s-api-ml

# Docker & Kubernetes API for ML Inference

Цей проєкт демонструє контейнеризацію машинного навчання (ML Inference API) за допомогою **Docker** та його розгортання й оркестрацію в кластері **Kubernetes**. API побудовано на базі сучасного асинхронного фреймворку для Python — **FastAPI**.

## 🚀 Основний технологічний стек
* **Python** — основна мова розробки та роботи з ML-моделями.
* **FastAPI** — високопродуктивний веб-фреймворк для створення REST API з автоматичною інтерактивною документацією (Swagger UI / ReDoc).
* **Docker** — контейнеризація додатку для забезпечення ідентичності середовища розробки та продакшену.
* **Kubernetes (K8s)** — оркестрація контейнерів, забезпечення масштабованості, відмовостійкості та керування ресурсами.
* **Pandas / Scikit-learn / NumPy** — бібліотеки для передобробки даних та виконання інференсу моделі.

## 📂 Структура проєкту
docker-k8s-api-ml/
├── app/
│   ├── main.py          # Головний файл FastAPI додатку (маршрути, ініціалізація)
│   ├── model.py         # Логіка завантаження моделі та виконання інференсу (Prediction)
│   └── requirements.txt # Залежності Python-додатку
├── k8s/
│   ├── deployment.yaml  # Конфігурація Kubernetes Deployment (опис подів, реплік, контейнерів)
│   └── service.yaml     # Конфігурація Kubernetes Service (балансувальник навантаження / доступ до API)
├── Dockerfile           # Інструкція для збірки Docker-образу додатку
├── docker-compose.yml   # Скрипт локального запуску контейнерів (опціонально)
└── README.md            # Документація проєкту

Перевірка статусу розгортання:
# Перевірка статусів Pods
kubectl get pods
# Перевірка статусів Deployments
kubectl get deployments
# Отримання інформації про Service та доступні порти
kubectl get services
